## 완료된 작업

### 1. Enemy FSM 리팩터링
- Boolean gate → 공식 FSM (EnemyState: IDLE/ATTACK/EVADE/HIT/DEAD)
- ATTACK sub-phase: STARTUP→ACTIVE→RECOVERY (animation frame-driven)
- `transition_to()` 단일 창구, `_enter_state()` / `_exit_state()` 생명주기
- ADR: `docs/adr/0001-enemy-fsm-architecture.md`
- Spec: `docs/superpowers/specs/2026-05-15-enemy-fsm-design.md`
- 테스트: `tests/unit/test_enemy_fsm.gd` (23개)

### 2. Godot 디버거 경고 수정
- SHADOWED_VARIABLE_BASE_CLASS, INTEGER_DIVISION, STATIC_CALLED_ON_INSTANCE 등 전부 수정

### 3. GameState 모듈 정리
- `game_constants.gd` 생성 (단일 진실 공급원)
- dead constants/functions 제거
- delegation 계약 테스트 추가 (37개)

### 4. ML 파이프라인
- 단일/2프레임/4프레임 모델 비교 완료
- **2프레임 시퀀스(Conv1D+LSTM)가 기본** (99.94%~100%)
- Gaussian noise (σ=0.03) + 스케일 ±20% 증강 적용
- GPU 가속 토글 추가 (설정→웹캠)
- 2프레임 모델 우선 로드

### 5. 성능 최적화
- GUARD_EXIT_FRAMES 5→2 (가드 해제 66ms)
- Per-side cooldown (L→R→L 콤보 허용)
- `UDP_PUNCH_TIME_SCALE` 0.42→0.30
- `min_tracking_confidence` 0.5→0.7

### 6. 버그 수정
- Training respawn KO callback leak
- Guard + punch animation desync (`_cancel_guard()`)
- Settings panel tabs broken (name→content_name)

### 7. 코드 정리
- `dodge` → `squat` 완전 통일
- `scratch/`, `pose_ml_backup/`, `conversation_history.md` 등 삭제
- Git history 초기화 (`rm -rf .git && git init`)
- README.md 최신화
